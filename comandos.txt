comandos:

    'git init'  => cria uma pasta .git no repositorio que inicia o versionamento do projeto
    'git status' => mostra as alterações feitas naquele momento
    'git add <path>' => adiciona os arquivos para subir
        exemplo 1 : 'git add .'    ** sobe tudo dentro do path
        exemplo 2 : 'git add comandos.txt'  ** sobe um unico arquivo especificado
        exemplo 3 : 'git add pasta/teste.py' ** sobe um unico arquivo, dentro de uma pasta especificada
    'git commit -m <mensagem a ser escrita>'
    'git push'
    'git push'
